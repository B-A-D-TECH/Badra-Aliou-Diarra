# Badra Aliou Diarra — Full-Stack Developer Portfolio

This portfolio has been upgraded into a modern React + Vite application with a professional layout, polished sections, and a responsive experience.

## Features
- React + Vite frontend for fast development and build performance
- Responsive portfolio sections for hero, projects, skills, about, and contact
- Theme toggle, mobile navigation, and polished visual design
- Built with TypeScript and clean CSS styling

## Local Development
Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
